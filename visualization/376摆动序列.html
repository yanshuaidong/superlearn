1. 解题思路模块

目标：找到最长摆动子序列长度。摆动要求相邻差值符号交替变化（正负正负或负正负正）。

贪心核心：只在“趋势发生变化”时计数。我们不需要记录整个子序列，只需要看当前差值 curDiff 和上一段差值 preDiff 是否一正一负。

当满足 (curDiff > 0 && preDiff <= 0) || (curDiff < 0 && preDiff >= 0) 时，说明出现新的拐点，可以把当前数接进摆动序列，长度 result++，并更新 preDiff = curDiff。

趋势变化可视化：哪些点计数，哪些点不计数

计数：发生“上升 ↔ 下降”切换的拐点

折线方向每次反转（峰或谷）都是有效摆动点，计数。

不计数：仍在同一方向，或差值为 0

一直上升/一直下降只是同一趋势延续；平坡（差值 0）也不形成摆动，不计数。

绿色点：计数点（拐点） 红色点：不计数点 灰色线：趋势延续
时间复杂度 O(n)，空间复杂度 O(1)
