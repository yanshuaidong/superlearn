1143. 最长公共子序列

Longest Common Subsequence — 动态规划经典二维 DP

📋题目描述

给定两个字符串 text1 和 text2，返回这两个字符串的最长公共子序列的长度。如果不存在公共子序列，返回 0。

子序列是指从原字符串中删除一些字符（也可以不删除）后，剩余字符保持原有顺序组成的新字符串。 例如，"ace" 是 "abcde" 的子序列，而 "aec" 不是。

示例 1：
输入：text1 = "abcde", text2 = "ace"
输出：3
解释：最长公共子序列是 "ace"，长度为 3
示例 2：
输入：text1 = "abc", text2 = "def"
输出：0
解释：两个字符串没有公共子序列

🧠核心思路

用 dp[i][j] 表示 text1 的前 i 个字符与 text2 的前 j 个字符的最长公共子序列长度。

状态转移方程：

① 若 text1[i-1] == text2[j-1]：
   dp[i][j] = dp[i-1][j-1] + 1 （两字符相同，接到已有的 LCS 后面）

② 若 text1[i-1] != text2[j-1]：
   dp[i][j] = max(dp[i-1][j], dp[i][j-1]) （舍弃某一个字符，取更优结果）

💻答案代码（带注释）

/**
 * 1143. 最长公共子序列
 * @param {string} text1
 * @param {string} text2
 * @return {number}
 */
var longestCommonSubsequence = function(text1, text2) {
    // text1 的长度 m，text2 的长度 n
    const m = text1.length;
    const n = text2.length;

    // 创建 (m+1) × (n+1) 的二维数组，全部初始化为 0
    // 多一行一列是为了处理"某个字符串为空"的边界
    const dp = Array.from({ length: m + 1 },
                        () => Array(n + 1).fill(0));

    // 从 i=1, j=1 开始填表
    for (let i = 1; i <= m; i++) {
        for (let j = 1; j <= n; j++) {

            if (text1[i - 1] === text2[j - 1]) {
                // 字符相同 → 在"左上角"基础上 +1
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                // 字符不同 → 舍弃其中一个字符，取较大值
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }

    // dp[m][n] 就是两个完整字符串的最长公共子序列长度
    return dp[m][n];
};
