53. 最大子数组和

动态规划 · 连续子数组 · 最优子结构

📋 题目描述

给你一个整数数组 nums，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

示例：
输入：nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
输出：6
解释：连续子数组 [4, -1, 2, 1] 的和最大，为 6。

💡 核心思路

dp[i] 的含义：以 nums[i] 结尾的连续子数组的最大和。

状态转移方程：

dp[i] = max( nums[i], dp[i-1] + nums[i] )

每到一个新元素，就问自己一个问题：
"前面累加的和还有用吗？"
如果 dp[i-1] > 0 → 前面有正贡献，接上它更好
如果 dp[i-1] ≤ 0 → 前面是累赘，不如从当前重新开始

🧑‍💻 答案代码（带注释）

/**
 * 53. 最大子数组和
 * 动态规划解法
 */
var maxSubArray = function(nums) {

    // dp[i] 表示：以 nums[i] 结尾的连续子数组的最大和
    let n = nums.length;

    // 创建 dp 数组，长度为 n
    let dp = new Array(n);

    // 🟢 初始化：第一个元素只能"自己组成子数组"
    dp[0] = nums[0];

    // 🟡 maxSum 追踪全局最大值（最终答案）
    let maxSum = dp[0];

    // 从 i = 1 开始逐个决策
    for (let i = 1; i < n; i++) {

        // 🔵 核心转移：接上前面 or 自己重新开始
        dp[i] = Math.max(
            nums[i],               // 选择A：从自己重新开始
            dp[i - 1] + nums[i]   // 选择B：接在前面后面
        );

        // 🟡 随时更新全局最大
        maxSum = Math.max(maxSum, dp[i]);
    }

    return maxSum;
};

⏱ 复杂度分析

时间复杂度：O(n) — 只需遍历数组一次

空间复杂度：O(n) — dp 数组的大小（可优化到 O(1)）
