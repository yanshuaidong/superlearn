## 大纲（爽文+智谋｜明线诛阉党，暗线对皇太极）

### 一句话卖点
朱由检登基前夜遭伏杀——他先不谈救国，只做一件事：**三日夺刀**（夺宫门之刀、夺厂卫之刀、夺钱粮之刀）。明面上他“依法钦定逆案”，暗地里他要截断一条更致命的线：**魏忠贤与皇太极的私通**。

### 总基调（正文写作的镜头规则）
- **冷峻宫廷 + 快节奏智斗**：对白短、动作快；关键处用“细节证据”压人（手茧、印章、门禁、轮值册、口供时间线）。
- **爽点不靠吹**：每次出手都讲“流程/证据链”，读者爽在“对手以为稳了→突然被规则碾死”。
- **双线并行**：明线每章推进；暗线每2-3章回响一次“关外鼓点”，越敲越近。

### 明线 / 暗线（写作导航）
- **明线（诛阉党）**：控卫换血 → 断联络 → 立案取证 → 三司会审 → 钦定逆案 → 抄家充公（解决财政第一口血）。
- **暗线（魏×皇太极）**：刺杀失败 → 密信外逃 → 关外得报、加速筹谋 → 主角在朝内清洗时，被迫同步布局边防与粮饷。

### 关键人物（可随写作增删）
- **主角/新君（朱由检）**：现代人思维，懂“系统性崩塌”，做事重流程与证据链；不喊口号，先把刀握在手里。
- **张皇后**：稳、清醒；给主角“名分台阶”，也给自己“后宫自保”。
- **魏忠贤**：表面哭到肝胆俱裂，背地里每滴眼泪都是刀；退意是假，求生与翻盘是真。
- **王承恩**：御前心腹，负责“近身安全+口谕落地”，主角把命交出去的那个人。
- **吴孟明**：阉党旧仇，适合做“暗处的刀”；办事狠，敢越线。
- **李若琏**：稳、能扛；适合做“明面上的刀”，拿圣旨换岗、追杀、封口。
- **温体仁**：老狐狸嗅到风向变化，既想自保又想押注（适合写一场“眼神戏”）。

### 四个“靶子”（抄家回血 + 打穿利益网）
> 写成“当代后人/门生故吏一系”，避免时代错位，也符合你“栽赃后代也行”的设定。
- **A｜京畿勋戚田庄**：侵田、强买、私设庄兵；和宫门守卫/京营有暗线。
- **B｜侯伯府系吞并网**：牵连地方官；盐课/粮运里吃空，账册最脏。
- **C｜董其昌（或其门生门客一系）**：名望护体，田契来路不净；舆论与“清名”是他们的甲胄。
- **D｜“徐阶门生故吏/后裔”一系（或虚构徐某勋戚）**：写的是“当代承袭的势力与田产链”，不写徐阶本人。
- **备选加钉**：**诚意伯刘基之后**一支（“开国功勋之后”的免死感更好写对抗）。

## 第一卷：三日夺刀（1—9章）
> 这一卷只干一件事：**登基前后 72 小时，把刀从别人手里夺回来**。每章都要落到“控卫/控人/控证据/控钱”其中一项。

### 第1章｜西华门夜雨（入局+伏杀）
- **画面骨架**：西华门奉诏入宫（车中摸到内衬硬物：防弹背心）→ 文华殿烛影摇、天启弥留喘息 → 张皇后一语镇住满殿风声（主角一瞬恍惚：眉眼竟与前世恋人影子重叠）→ 魏忠贤伏地痛哭“奉遗命” → “重用魏”四字像钉子钉进殿梁 → 遗诏宣读、名分移交 → 新君出宫返府，巷口忽起寒光，刀取喉胸；软甲挨一刀救命 → 火把骤亮，覆面锦衣卫清场封口。

### 第2章｜信王府落闩（底牌亮规矩）
- **画面骨架**：信王府门闩落下、灯火加倍却更像牢笼 → 周氏扑来强忍发抖，盯见衣襟撕裂几欲失声 → 主角点到即止复盘宫中“遗命”与巷口伏杀 → 书房关门亮底牌：拼夕夕银兑、五里投放、可买台灯/电脑/打印机（唯独缺火器）→ 台灯冷白光下键盘疾响，打印机吐出“方略表”。



### 第3章｜奉天门山呼（登基日不走常规早朝）
- **画面骨架**：

第一幕：告祭天地/宗庙，钟鼓声像催命 → 奉天门受朝贺，百官山呼却不敢抬眼 → 魏忠贤站位微妙，隐藏在了队伍最后面（且表情微妙，是那种昨天安排了人杀皇帝，但是没杀成的心理） → 大典毕。

第二幕：新君开始早朝。新皇帝把应该做的事情做了，就是立皇后，和皇太后等一众事情。随后魏忠贤递“告老还乡”的退意奏疏，新皇帝没有顺着魏忠贤的话说同意还不同意，表态先不说这个事情，赏赐魏忠贤去守孝。等于把魏锁进灵堂。不让魏忠贤和外界沟通。


### 第4章｜御书房两把刀（认刀+授刀）
- **画面骨架**：王承恩先查“身后无尾巴” → 御书房门一合，空气都紧了 → 吴孟明入内，声音一出，主角用“呼吸节奏/口音”认出：巷口那人 → 主角不问来历，只问一句：“你敢不敢用你的命，换朕的三天？” → 再召李若琏，主角盯他左手老茧与刀鞘磨痕，点破其参与护驾。
- **本章推进目标**：建立“明刀（李）暗刀（吴）”双系统；下令：换岗、封口、扣档、截信。
- **章尾钩子**：魏忠贤欲递消息，被吴孟明当场按住；魏把纸团吞下去，喉结一滚，像吞下一条毒蛇。

早朝上完毕，朱由检（皇帝）回到养心殿，开始布局自己的计划。
先是让王承恩秘密找来锦衣卫吴孟明。
吴孟明早年袭锦衣千户，佐许显纯处理北司刑。天启初年，因得罪许显纯被贬为锦衣卫小旗，家人也被迫害。许显纯是魏忠贤的爪牙。专门残害忠良。
吴孟明过来的途中也是有心事，莫非救皇帝朱由检的事情暴露了？心中有疑惑。
不久吴孟明，就进入养心殿。
吴孟明说了一句参见陛下。
朱由检立马就通过声音辨别出来了，他就那天夜晚救他的锦衣卫。
当即点破。
吴孟明大惊，没想到新皇帝朱由检记忆力如此好。于是承认，并说张皇后有恩于自己，自己偷偷在搜集魏忠贤的罪证，被魏忠贤的爪牙许显纯发现了，要杀自己，是张皇后救下了自己。如今自己的家人都被魏忠贤杀死，就剩下他一个人了。
吴孟明情绪激动，把自己收集到的证据呈递给朱由检。并恳请皇上处置专权跋扈的魏忠贤。
朱由校心想，刚想要睡觉就有人送枕头。
就立马洗漱为魏忠贤的十大罪状，并提拔吴孟明为锦衣卫指挥同知。
并立即下派任务，按照名单，将魏忠贤在东厂西厂锦衣卫各处的同党全部压入锦衣卫大狱。
吴孟明立刻接收旨意。
吴孟明出门后查看朱由检给的名单，竟然和自己潜伏5年收集的名单人名和罪状一模一样，暗自感叹，新皇帝圣明，早就在收集魏忠贤党派和爪牙了。
安排完一件事，朱由检又叫王承恩去叫李若琏。
李若琏，上林苑番育署人。天启六年武进士，曾经官至锦衣卫指挥使，1644年李自成攻破北京城后，在家的李若琏自杀殉国。
在朱由检翻阅的资料里面李若琏最有能力，且最为忠心。
李若琏进到养心殿后，朱由检观察到李若琏左手有老茧，是左撇子。
于是诈李若琏是否是也在那天夜晚，救了自己。
李若琏也是当场震惊，并回答是。
于是再问当天同行的人还有谁。李若琏回答了几个名字，其中就有吴孟明。
李若琏疑惑皇上是如何看出来的。
自然是推理出来的，你左手有茧子，且是刀兵的茧子，同时又是锦衣卫，惯用左手的人又缩小了。我方才就是诈一下你，你就承认了。
李若琏表示确实是惯用左手兵器，因为右手受伤过。
朱由检大笑，不愧是武进士，武学造诣甚高。
最后话锋一转，说朕今日照你来是让你来帮朕清理专权跋扈的魏忠贤的。
你敢不敢。
李若琏表示只忠心皇帝一人，愿意成为皇帝的刀兵。
朱由校很是高兴。让李若琏干两件事情。
第一件事情，按照名单上的名字。将北京城的禁卫军，羽林卫，三大营，五军营，三千营，神机营。的魏忠贤爪牙换掉压入锦衣卫狱。并撤换皇城守卫官军，留守卫，五城兵马司的人。同时严防魏忠贤勾结的贪官转移家属和财产。
第二件事情。比较机密，需要你（李若琏）找十个身手好且信得过的手下，把栽赃魏忠贤的罪证放到京城四大家族里面。
这四大家族分别是明朝嘉靖年间的内阁首辅徐阶其来的徐家，如今是他孙子徐朗在朝为官。和开国勋臣刘基之后刘世延刘家等家族。
这四大家族侵占民田，强买强卖，土地兼并达百万亩之巨。实乃大明朝最大的虫豸。
此次要借魏忠贤来借刀杀人，搂兔子打草。将他们一并处置。
李若琏这第二件事千万不能泄密。
李若琏赶紧表忠心。并称赞皇上圣明。大明朝有救了。











### 第5章｜宫门换血（把刀从宫门夺回来）
- **画面骨架**：以“登基安防”为由，宫门值守、内廷近侍、锦衣卫轮值一夜换尽 → 北镇抚司“查档”名义接管要害处 → 东厂线人名册与往来口供先扣后审。
- **本章推进目标**：断联络，让阉党变“瞎子+聋子”；开始搭证据链（口供时间线、轮值册、收支账、印信流转）。
- **章尾钩子**：城门夜色里，一个小兵从吊篮滑出城墙；李若琏追出十余里未及，只捡到一截断绳——**信，已经出城。**

### 第6章｜关外回音（暗线第一次露齿）
- **画面骨架**：密信经驿站、商队、边军手里一层层换手 → 雪夜里，关外营帐灯火不灭 → 皇太极拆信，读到“刺杀失败”四字，指节敲案，笑意薄得像刀。
- **本章推进目标**：把“刺杀”升级为“战争前奏”，让读者明白：朝内清洗不是终点，是抢时间。
- **章尾钩子**：皇太极回一句话：**“既杀不死，就让他忙死。”**

### 第7章｜逆案起（证据链开闸）
- **画面骨架**：大理寺/都察院/锦衣卫互相试探 → 主角用打印机做“统一笔迹”的假信做引线（不直接定罪，用来逼人露出真实链条）→ 魏忠贤被逼到只能“合作换命”，却暗藏翻供坑。
- **本章推进目标**：从“抓人”转成“立案”：口供、账册、轮值、印信、田契五线互咬。
- **章尾钩子**：魏冷笑：要死一起死，我要拉东林下水。朝堂风声顿起。

### 第8章｜三司会审（法的刀落下去）
- **画面骨架**：公堂上乌压压的人，谁都怕被点名 → 魏口供像乱箭，专射“清流名士” → 大理寺搜证反而搜出“四家”铁证 → 关键一击：对照字迹/印鉴/田契来源，“清名”当场裂开。
- **本章推进目标**：完成“钦定逆案”的合法性：不是皇帝情绪，是证据与法条。
- **章尾钩子**：钱嘉徵“十大罪”呈上；主角只回四字：**“依法办理。”**

### 第9章｜抄家封门（财政第一口血）
- **画面骨架**：一张圣旨换掉一座城的夜色 → 封条贴上侯府门环，哭声从院里漏出来 → 银锭堆出冷光，田契一摞摞像尸骨 → 百姓围观，第一次觉得“皇帝的刀”砍到了该砍的人。
- **本章推进目标**：阶段性结算：银两、田地入库；主角获得“做下一步改革”的燃料。
- **章尾钩子**：户部摊开账本：国库亏空像深井。内阁有人嘲笑“抄家救不了天下”，主角却提起另一桩旧事：**“成祖当年名号，朕想改。”**

### 内阁赌局｜税收翻倍（用“改太宗”当诱饵）
