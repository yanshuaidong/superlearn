崇祯帝朱由检在位共17年（**1628年—1644年**），这一时期明朝内忧外患交织，天灾人祸频发，最终走向覆灭。以下是按年份梳理的崇祯朝每年重大事件：

### 崇祯元年（1628年，戊辰）
1.  **清算阉党**：崇祯帝继位后，迅速铲除魏忠贤阉党集团，魏忠贤自缢身亡，客氏被处死，下诏定“逆案”，清洗阉党余孽，朝野为之一振。
2.  **裁撤驿站**：为节省财政开支，下令裁撤全国驿站，大量驿卒失业，其中就包括**李自成**，成为明末农民起义的导火索之一。
3.  **陕北民变爆发**：陕西、陕北地区遭遇严重旱灾、蝗灾，官府催征赋税严苛，王嘉胤、王左桂等率先举起义旗，揭开明末农民大起义的序幕。
4.  **后金犯边**：后金皇太极派兵袭扰辽东边境，明朝辽东防线压力陡增。

### 崇祯二年（1629年，己巳）
1.  **己巳之变**：皇太极率领后金大军绕道蒙古，突破长城喜峰口，直逼北京，明朝京师戒严。
2.  **袁崇焕入狱**：辽东督师袁崇焕率部驰援北京，击退金军，但崇祯帝中皇太极“反间计”，以“通敌叛国”罪将袁崇焕下狱。
3.  **农民起义蔓延**：高迎祥、张献忠等先后加入起义军，起义势力从陕北扩散至山西。

### 崇祯三年（1630年，庚午）
1.  **袁崇焕被凌迟**：袁崇焕被处以凌迟之刑，辽东军心涣散，明朝辽东防御体系逐渐崩坏。
2.  **定逆案收官**：朝廷公布“逆案”名单，共惩处阉党成员258人，阉党势力基本肃清。
3.  **孙承宗督师辽东**：重新启用孙承宗，收复永平、遵化等四城，暂时稳住辽东局势。
4.  **起义军壮大**：王嘉胤率军攻克府谷，李自成投奔其部下，农民起义军规模进一步扩大。

### 崇祯四年（1631年，辛未）
1.  **大凌河之战**：后金围攻大凌河城，守将祖大寿粮尽援绝，被迫诈降后金，明朝关外精锐损失惨重。
2.  **孔有德叛乱**：登州参将孔有德因军饷拖欠、上司欺压，发动叛乱，攻占登州，掳掠大量粮草、军械。
3.  **天灾加剧**：陕西、山西持续旱灾、蝗灾，百姓流离失所，纷纷加入起义军。

### 崇祯五年（1632年，壬申）
1.  **孔有德陷莱州**：孔有德叛军围攻莱州，明朝调集重兵围剿，久攻不下，山东局势危急。
2.  **农民军入晋**：起义军主力从陕西进入山西，连克州县，张献忠率军攻破隰州，声势浩大。
3.  **后金扩军备战**：皇太极整合后金各部，仿制红衣大炮，增强军队攻坚能力。

### 崇祯六年（1633年，癸酉）
1.  **孔有德降后金**：孔有德、耿仲明率叛军及水师、红衣大炮投降后金，后金军事力量大幅提升，掌握火器优势。
2.  **农民军渡黄河南下**：起义军突破明军包围，从山西渡过黄河，进入河南，势力扩展至中原腹地。
3.  **旅顺陷落**：后金出兵攻打旅顺，明朝守将黄龙战死，旅顺失守，辽东半岛防御崩溃。

### 崇祯七年（1634年，甲戌）
1.  **陈奇瑜围剿农民军**：朝廷任命陈奇瑜为五省总督，统一指挥陕、晋、豫、川、湖广军务，围剿起义军。
2.  **车厢峡之困与诈降**：起义军误入陕西车厢峡，被明军包围，粮草断绝。李自成、张献忠等用诈降计，假意投降，突围后迅速复叛，明军围剿计划破产。
3.  **清军首次入塞侵扰宣大**：皇太极率军攻打宣府、大同，劫掠大量人口、财物后撤军。

### 崇祯八年（1635年，乙亥）
1.  **荥阳大会**：13家72营起义军首领在河南荥阳召开会议，李自成提出“**分兵定向，四路攻战**”策略，统一起义军作战方向，是明末农民起义的重要转折点。
2.  **焚毁凤阳皇陵**：高迎祥、李自成率军攻克明朝中都凤阳，焚毁皇陵和龙兴寺，崇祯帝痛哭流涕，下罪己诏。
3.  **明军调整围剿策略**：崇祯帝震怒，处死多名官员，任命洪承畴为五省总督，全力围剿起义军。

### 崇祯九年（1636年，丙子）
1.  **皇太极称帝，改国号为清**：皇太极在盛京称帝，改国号“大金”为“大清”，改元崇德，正式建立清朝，与明朝分庭抗礼。
2.  **高迎祥被俘牺牲**：高迎祥率军攻打西安，在盩厔（今陕西周至）遭遇明军伏击，战败被俘，后被凌迟处死。**李自成继称“闯王”**，成为起义军核心领袖。
3.  **清军第三次入塞**：清军绕道蒙古，攻入长城，侵扰京畿、河北等地，掳掠人口十余万。

### 崇祯十年（1637年，丁丑）
1.  **杨嗣昌提出“四正六隅，十面张网”**：兵部尚书杨嗣昌制定围剿农民军的战略，将全国划分为“四正六隅”十个战区，企图将起义军困死、歼灭。
2.  **张献忠谷城诈降**：张献忠在湖北谷城向明军投降，接受招安，但拒绝解散军队，暗中积蓄力量。
3.  **加征“剿饷”**：为支撑围剿军费，朝廷加征剿饷280万两，百姓负担进一步加重。
4.  **李自成潜伏商洛山**：李自成在潼关南原遭遇明军重创，率残部18骑逃入商洛山，蛰伏待机。

### 崇祯十一年（1638年，戊寅）
1.  **清军第四次入塞**：多尔衮、岳托率领清军分兵两路入塞，深入山东，攻克济南，杀德王朱由枢，掳掠人口、财物无数，明军疲于奔命。
2.  **农民起义陷入低潮**：洪承畴、孙传庭率军围剿，起义军损失惨重，李自成隐匿商洛山，张献忠受困谷城，起义暂时转入低潮。
3.  **加征“练饷”**：为训练边军，朝廷加征练饷730万两，与之前的辽饷、剿饷合称“**三饷加派**”，百姓不堪重负，流民日增。

### 崇祯十二年（1639年，己卯）
1.  **张献忠谷城复叛**：张献忠在谷城重举义旗，罗汝才等部纷纷响应，农民起义再次爆发高潮。
2.  **杨嗣昌督师剿贼**：崇祯帝任命杨嗣昌为督师，统领全国明军围剿张献忠，双方在湖广、四川展开拉锯战。
3.  **清军撤军**：清军在关内劫掠近一年后，携带大量战利品撤军，明军尾随追击，伤亡惨重。

### 崇祯十三年（1640年，庚辰）
1.  **中原大灾荒**：河南、陕西、山东等地爆发特大旱灾、蝗灾，“人相食”的惨状遍及各地，李自成趁机从商洛山复出，进入河南。
2.  **李自成提出“均田免赋”**：李自成提出“均田免赋”“迎闯王，不纳粮”的口号，迅速得到百姓拥护，起义军规模扩张至数十万。
3.  **杨嗣昌入川围剿张献忠**：杨嗣昌率军入川，企图将张献忠围歼于四川，张献忠采取“以走制敌”策略，拖垮明军。

### 崇祯十四年（1641年，辛巳）
1.  **李自成克洛阳，杀福王**：李自成率军攻克洛阳，处死福王朱常洵，开仓放粮，赈济百姓，起义军声势震天。
2.  **张献忠克襄阳，杀襄王**：张献忠率军奔袭襄阳，处死襄王朱翊铭，杨嗣昌听闻后，畏罪自杀于夷陵。
3.  **松锦大战爆发**：洪承畴率领8总兵、13万明军救援锦州，与清军展开决战，明朝倾尽国力赌上关外最后精锐。
4.  **开封之战**：李自成率军围攻开封，明军顽强抵抗，双方伤亡惨重，开封城暂时得以保全。

### 崇祯十五年（1642年，壬午）
1.  **松锦大战惨败**：明军主帅洪承畴指挥失误，明军粮草被清军截断，全军崩溃，洪承畴被俘后降清，祖大寿再次降清。明朝关外精锐损失殆尽，**宁锦防线彻底崩溃**。
2.  **李自成三围开封**：李自成第三次围攻开封，明军为解围掘开黄河大堤，导致开封城被洪水淹没，数十万百姓丧生。
3.  **清军第五次入塞**：清军入关侵扰，深入河北、山东、苏北等地，掳掠人口百万，明军毫无抵抗之力。
4.  **孙传庭入狱**：兵部侍郎孙传庭因反对杨嗣昌策略，被崇祯帝下狱，明朝失去一员能战大将。

### 崇祯十六年（1643年，癸未）
1.  **孙传庭战死潼关**：崇祯帝重新启用孙传庭，任命其为督师，率秦军出关围剿李自成。双方在郏县、汝州决战，明军惨败，孙传庭战死，**明朝再也无力组织大规模围剿**。
2.  **李自成建大顺政权**：李自成率军攻克西安，改西安为西京，建立“大顺”政权，年号永昌，整顿军队，准备进军北京。
3.  **张献忠建大西政权**：张献忠率军攻克武昌，处死楚王朱华奎，建立“大西”政权，后率军入川，准备建立根据地。
4.  **左良玉拥兵自重**：明军将领左良玉手握数十万大军，盘踞武昌，不听朝廷调遣，明朝中央权威彻底瓦解。

### 崇祯十七年（1644年，甲申）
1.  **李自成北伐北京**：李自成率领大顺军从西安出发，渡过黄河，攻克太原、大同、宣府，明军守将纷纷投降，大顺军直逼北京。
2.  **崇祯帝调吴三桂入卫**：崇祯帝紧急下令调辽东总兵吴三桂率军入卫北京，但吴三桂拖延行军，错失时机。
3.  **北京陷落，崇祯自缢**：三月十九日，大顺军攻克北京外城，崇祯帝朱由检在煤山（今景山）自缢身亡，时年34岁，**明朝灭亡**。
4.  **吴三桂引清兵入关**：吴三桂投降清朝，引清兵入关，在山海关一片石大战中击败李自成大顺军。
5.  **清军入主中原**：清军击败大顺军后，进入北京，开始确立对全国的统治。
